## 4. 是否发 1.1.x 的判断

发 1.1.x：

- 任一 P0 可复现。
- 审核反馈要求必须改。
- 多个用户在同一核心链路卡住，例如不能打开、不能加入、不能生成清单、分享卡片不能回传。
- 登录失败且游客不能继续。

先观察，不急着发版：

- 推荐口味不够准，但用户仍能换一组或自己挑。
- 文案不够清楚，但用户能完成菜单/清单。
- 单个设备的偶发卡顿，暂不可复现。
- 普通视觉、做法细节、性能建议。
