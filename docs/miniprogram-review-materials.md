本文档用于填写微信公众平台和提交审核。正式提交和发布步骤见 `docs/miniprogram-platform-submit-runbook.md`；后台实时要求仍以微信公众平台为准。

## 1. 基本信息

### 小程序名称

Humi

### 简称

Humi

### 一句话介绍

帮家里决定今晚吃什么，并自动整理买菜清单。

### 详细介绍

Humi 是一款家庭晚饭决策与协作工具。用户可以打开后直接获得“今晚吃什么”推荐，也可以发起“问问大家想吃啥”，让家人通过小程序卡片免登录点一个感觉。主厨选择要做的菜后，系统会自动生成今晚菜单和买菜清单。

Humi 不要求用户维护库存或填写软口味表；忌口作为硬约束由用户主动填写，家里的饭线索主要来自感觉征集、想吃池子、买菜认领和晚间轻确认。

Humi 提供的是生活安排和信息整理能力，不提供医疗诊断、治疗建议或专业营养咨询。

## 2. 服务类目候选

最终类目必须以微信公众平台当前可选项和主体资质为准，不在提交前凭文档猜定。

建议按以下顺序检查：

1. 工具类中与生活、效率或信息管理相关的类目。
2. 生活服务中与家政、家庭生活相关的类目。
3. 餐饮相关类目仅在后台要求且主体资质满足时选择。

避免把 Humi 描述为：

- 医疗健康诊断工具。
- 营养治疗或疾病管理服务。
- 外卖、食品销售或到店餐饮服务。

## 3. 审核演示账号

首发核心体验不要求手机号。小程序不以邮箱作为主登录方式；微信登录用于账号保存和恢复，手机号绑定仅在“我的家/账号安全”中由用户主动点击授权，用于账号绑定、登录验证和找回。

若审核后台强制填写账号，填写“无需账号，打开即可体验核心功能”。如平台仍要求提供账号，再单独创建审核专用账号，不使用维护者私人账号。

## 4. 审核备注

可直接粘贴以下内容：

```text
Humi 是一款家庭晚饭安排工具，核心功能无需注册即可体验。

建议审核路径：
1. 打开小程序，进入【今晚】；
2. 点击“今晚就做”把推荐加入今晚菜单；
3. 点击“换一组”或“不想吃”查看推荐反馈；
4. 点击“问问大家想吃啥”查看家庭感觉征集入口；
5. 进入【清单】查看自动生成的食材清单；
6. 进入【我的家】查看家庭成员、饭线索、想吃池子、忌口和隐私协议入口。

推荐服务不可用时，页面会自动使用本地规则给出另一组菜单，不影响核心体验。
本产品仅提供家庭菜单、采购清单和协作整理，不提供医疗诊断或专业营养建议。
```

## 5. 隐私保护指引填写对照

后台选项名称可能随平台调整，应以实际页面为准。按最接近的选项如实选择。

| 产品数据 | 是否收集 | 使用目的 | 触发方式 |
| --- | --- | --- | --- |
| 微信身份标识 | 登录时 | 账号登录、恢复会话和后续家庭协作 | 小程序登录时 |
| 手机号 | 可选绑定时 | 账号绑定、登录验证、账号找回、家庭协作安全 | 用户在“我的家”主动点击绑定并授权 |
| 邮箱账号 | 小程序普通用户不收集 | 开发测试和海外预留 | 不在普通用户入口展示 |
| 密码 | 小程序普通用户不收集 | 开发测试和海外预留 | 不在普通用户入口展示 |
| 家庭成员邮箱 | 小程序首发不收集 | 后续家庭协作预留 | 不在普通用户入口展示 |
| 家庭成员数量 | 加入家庭时 | 调整份量和菜单建议 | 根据家庭成员关系形成，不要求单独填表 |
| 忌口、过敏 | 可选 | 作为硬约束避开不合适的菜 | 用户主动填写，也可跳过 |
| 今晚菜单、三餐轻记录和连排计划 | 使用功能时 | 保存和同步饮食安排 | 用户主动操作 |
| 食材清单、勾选状态和买菜认领 | 使用功能时 | 生成、保存和协作采购清单 | 用户主动操作 |
| 后台已有食材信号 | 使用功能时 | 推荐时作为加分项和清单去重参考 | 由清单勾选、做饭确认或轻确认生成 |
| 感觉征集、想吃池子和晚间反馈 | 使用功能时 | 形成家庭饭线索并改善后续推荐 | 用户主动操作 |
| 推荐反馈 | 换一组时 | 调整后续推荐 | 用户主动选择 |
| 使用事件、随机会话标识 | 使用功能时 | 统计核心流程、发现故障 | 自动生成，不读取微信身份 |

当前不收集：

- 微信头像和昵称。
- 精确位置。
- 通讯录。
- 相册内容。
- 摄像头和麦克风。
- 支付信息。

若后续接入相册保存权限、订阅消息、支付或其他新能力，必须先更新隐私保护指引和隐私政策。

## 6. 数据处理说明

### 游客

- 菜单、计划、清单、后台已有和家庭画像主要保存在当前 WebView 的本地存储。
- 仍可能发送不含微信身份的产品使用事件。
- 用户清理微信缓存或 WebView 数据后，本地数据可能丢失。

### 登录用户

- 微信身份登录后，家庭数据可同步到云端。
- 数据按家庭空间隔离。
- 推荐请求仅发送完成推荐所需的菜谱候选、忌口硬约束、后台已有信号、历史安排和反馈。
- 家庭成员邮箱已从推荐上下文中移除，推荐服务只接收匿名成员编号。
