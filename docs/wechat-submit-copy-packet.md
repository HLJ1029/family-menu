## 3. 基础介绍

### 一句话介绍

```text
帮家里决定今晚吃什么，并自动整理买菜清单。
```

### 详细介绍

```text
Humi 是一款家庭晚饭决策与协作工具。用户可以打开后直接获得“今晚吃什么”推荐，也可以发起“问问大家想吃啥”，让家人通过小程序卡片免登录点一个感觉。主厨选择要做的菜后，系统会自动生成今晚菜单和买菜清单。

Humi 不要求用户维护库存或填写软口味表；忌口作为硬约束由用户主动填写，家里的饭线索主要来自感觉征集、想吃池子、买菜认领和晚间轻确认。

Humi 提供的是生活安排和信息整理能力，不提供医疗诊断、治疗建议或专业营养咨询。
```

## 4. 审核备注

```text
Humi 是一款家庭晚饭安排工具，核心功能无需注册即可体验。

建议审核路径：
1. 打开小程序，进入【今晚】；
2. 点击“今晚就做”把推荐加入今晚菜单；
3. 点击“换一组”或“不想吃”查看推荐反馈；
4. 点击“问问大家想吃啥”查看家庭感觉征集入口；
5. 进入【清单】查看自动生成的食材清单；
6. 进入【我的家】查看家庭成员、饭线索、想吃池子、忌口和隐私协议入口。

推荐服务不可用时，页面会自动使用本地规则给出另一组菜单，不影响核心体验。
本产品仅提供家庭菜单、采购清单和协作整理，不提供医疗诊断或专业营养建议。
```

## 7. 隐私保护指引

应声明：

| 数据 | 用途 |
| --- | --- |
| 微信身份标识 | 账号登录、恢复会话、家庭协作 |
| 手机号 | 用户主动绑定时用于账号绑定、登录验证、账号找回、家庭协作安全 |
| 家庭成员数量 | 根据家庭成员关系调整份量和菜单建议，不要求单独填表 |
| 忌口、过敏 | 作为硬约束避开不合适的菜，用户可填写或跳过 |
| 今晚菜单、三餐轻记录、食材清单、买菜认领 | 保存和同步家庭吃饭安排与采购协作 |
| 感觉征集、想吃池子、晚间反馈 | 形成家庭饭线索并改善后续推荐 |
| 使用事件、随机会话标识 | 统计核心流程、发现故障 |

不声明：

- 精确位置。
- 通讯录。
- 相册内容。
- 摄像头、麦克风。
- 支付信息。
- 医疗健康数据。
