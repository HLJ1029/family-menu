2026-06-07

## 测试菜品
- 鱼香肉丝（肉菜，多食材混搭）
- 宫保鸡丁（肉菜，酱汁浓、花生脆感）

## 版本说明
- **Original**：旧 prompt — "Urban Family Kitchen 风格，米白纸感，细墨线勾边，柔和阴影"
- **New (V5)**：新 prompt — "水彩手绘美食插画，45°轻俯视，62-72% 主体占比，奶白 #F5F4F1 背景，禁止卡片/投影/阴影"

## 对比分析

| 维度 | Original | New (V5) | 变化 |
|------|----------|----------|------|
| 盘子完整性 | ✅ 完整 | ✅ 完整 | 持平 |
| 主体占比 | ⚠️ ~70% | ✅ ~70%（留白更稳定） | 微改善 |
| 背景颜色 | 🟡 米白/带暖黄晕染 | ✅ 奶白（#F5F4F1 方向） | 改善 |
| 卡片框 | ✅ 无 | ✅ 无 | 持平 |
| 投影/阴影 | ⚠️ 盘子/菜肴下有阴影 | ⚠️ 仍有轻微阴影 | 未完全消除 |
| 视角 | 俯视 | 俯视 | 持平 |
| 烟火气 | 🟡 偏干净统一 | 🟢 更浓、酱汁光泽更明显 | 改善 |
| 水彩质感 | 🟡 标准 | 🟢 笔触更自由、颜色更浓 | 改善 |

## 关键发现

### ✅ 成功改进
1. **背景色更稳定**：新 prompt 强制奶白背景，减少了旧版中的暖黄晕染
2. **烟火气更浓**：酱汁光泽和食材纹理更明显，颜色更浓
3. **卡片框/装饰**：两者都无卡片框，禁令有效
4. **盘子完整性**：两者都完整，V5 在鱼香肉丝上留白略好

### ⚠️ 仍有改进空间
1. **阴影未完全消除**：新版仍有轻微盘子底部阴影，需要加强"不要投影"的约束
2. **主体占比波动**：宫保鸡丁新版约 70%，但鱼香肉丝原版略满（约 75%+）
3. **一致性风险**：单次生成仍有波动，未来需要 few-shot reference 稳定风格

## 下一步建议

1. **采纳 V5 作为 baseline prompt** ✅
2. **迭代 V6**：在禁止列表中加入"不要盘子底部阴影"，强化留白约束
3. **添加 few-shot reference**：用 V5 的最佳样本作为 reference image，提升跨菜品一致性
4. **小批量重跑**：先用 3-5 个代表性菜品测试 V5+V6，达标后再全量重跑 43 张
